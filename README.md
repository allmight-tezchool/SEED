# googlesearch

競合サイトのGoogle検索流入数を自動推定するWebツール（AI抽出版）。

## 何ができるか

URLを入れるだけで以下を全自動で実行:

1. 競合サイトのHTMLを取得
2. **Claude Haikuがサイトの内容を読んで「狙ってそうなキーワード」を10個推測**
3. 各キーワードのGoogle検索結果から、対象サイトの順位を取得
4. 順位別CTRと推定検索数から月間Google流入数を推定

シングルURL分析と、複数URL比較（最大5サイト）に対応。

## セットアップ

`セットアップ手順書.md` を参照してください。GitHub + Vercel + Anthropic APIキーで18分で公開できます。

## 必要な環境変数

- `ANTHROPIC_API_KEY`: Claude APIキー（Vercelの環境変数として設定）

## 仕組み

- **キーワード抽出**: Claude Haiku 4.5にサイト情報（title・meta・h1〜h3・本文抜粋）を渡して「狙ってそうなキーワード10個」をJSON形式で出力させる
- **AIフォールバック**: Claude API失敗時はHTMLからの重み付きスコアリング方式に自動切替
- **順位取得**: User-Agentランダム化 + 1〜3秒のランダム待機 + ブロック検知時のリトライで、Google検索を直接スクレイピング
- **検索数推定**: AIが返す「高/中/低」ラベルから20,000/3,000/500回の概算値に変換
- **流入計算**: 順位別CTR（Advanced Web Rankingデータ参考） × 推定検索数 ÷ 捕捉率40%

## コスト

- Claude API: 1サイトあたり約1円（Haiku使用時）
- Vercel: 無料プランで運用可能
- Google検索: 無料（直接スクレイピング）

## 制約・注意

- 推定値です。誤差2〜3倍の可能性あり
- Google直接スクレイピング方式のため、1日20〜30サイトが安全圏
- 1サイト分析に25〜50秒かかる（AI推定＋待機を入れてブロックを回避するため）

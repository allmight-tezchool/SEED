# 🌱 SEED

つぶやきを書くと、AIが話題を広げて「知らなかった」を毎日届けてくれるアプリ。

---

## このアプリでできること

- Twitterのように一言つぶやくと、Claude(AI)がその話題を広げてくれる
- 関連する豆知識・別分野からの視点・もっと深掘りするキーワードを返してくれる
- すべて手元のデータベースに保存され、後から検索・振り返りできる

---

## ⚡ かんたん起動(おすすめ)

**`SEED起動.bat`** をダブルクリックするだけ。

- 初回: 自動で仮想環境を作って、必要なライブラリをインストールしてくれます(数分かかります)
- 2回目以降: 数秒でブラウザが開いて使えます
- 停止したいとき: 黒い画面で `Ctrl + C`、または **`SEED停止.bat`** をダブルクリック

> 💡 デスクトップにショートカットを作っておくと毎日が楽です。
> `SEED起動.bat` を右クリック → 「ショートカットの作成」 → デスクトップに移動

ただし、初回はステップ1とステップ4だけ手動で必要です:

- **ステップ1** (Pythonのインストール) … PCに Python が入っていない場合のみ
- **ステップ4** (APIキーの設定) … `.env` ファイルの作成

それ以外(仮想環境の作成、ライブラリのインストール、起動)は全部 `SEED起動.bat` がやってくれます。

---

## ステップ1: Python をインストールする

すでに Python 3.11〜3.13 が入っている方は飛ばしてOK。

> ⚠️ Python 3.14 は新しすぎてライブラリ(Pillowなど)が未対応のため、必ず 3.12 を使ってください。
> 既に 3.14 が入っていても共存できます。アンインストール不要。

### Windowsの場合(推奨: コマンド1行)

`py` ランチャー(Microsoft Store版 Python Install Manager)が入っていれば、コマンドプロンプトを開いて:

```
py install 3.12
```

これだけで Python 3.12 が入ります。

### 通常のインストーラーで入れる場合

1. https://www.python.org/downloads/release/python-31210/ にアクセス
2. ページ下部「Files」から **「Windows installer (64-bit)」** をダウンロード
3. インストーラー起動 → 以下を**両方**チェック:
   - ✅ Add python.exe to PATH
   - ✅ py launcher
4. 「Install Now」をクリック

確認のため、コマンドプロンプトで:
```
py -0
```
を打って `-V:3.12[-64]` が表示されればOK。

## ステップ2〜3(自動)

`SEED起動.bat` をダブルクリックすると、仮想環境の作成と必要なライブラリのインストールを自動でやってくれます。

---

## ステップ4: APIキーを設定する

1. このフォルダにある `.env.example` を **`.env`** という名前にリネーム
   (Windowsエクスプローラーで「拡張子を表示」を有効にしておくとやりやすい)
2. メモ帳などで `.env` を開いて、`sk-ant-xxxx...` の部分をご自身のAPIキーに書き換える
3. 保存して閉じる

> ⚠️ **重要**: `.env` ファイルは絶対にネットに公開しないこと(GitHubに上げないこと)。
> `.gitignore` に登録済みなので、Git管理しても自動で除外されます。

---

## ステップ5: 起動する

**`SEED起動.bat` をダブルクリック** するだけ。

自動でブラウザが開いて `http://localhost:8501` でアプリが表示されます。

> 終わるときは、ターミナルで `Ctrl + C`、または `SEED停止.bat` をダブルクリック。

## ファイル構成

```
SEED/
├─ app.py            ← Streamlitの画面(メイン)
├─ db.py             ← データベース処理(SQLite)
├─ ai.py             ← Claude APIを呼ぶ部分
├─ requirements.txt  ← 使うライブラリ一覧
├─ .env.example      ← APIキー設定の見本
├─ .env              ← (自分で作成) APIキー設定本体
├─ .gitignore        ← Git管理から外すファイル
├─ SEED起動.bat      ← ★ 1クリック起動ランチャー
├─ SEED停止.bat      ← 停止ランチャー
├─ 診断ツール.bat    ← トラブル時の環境診断
├─ seed.db           ← (起動後に自動生成される) データベース本体
└─ README.md         ← この手順書
```

## 将来Webアプリにするときのヒント

このコードはそのまま **Streamlit Community Cloud**(無料)にデプロイできます。

1. GitHubにこのフォルダをアップロード(`.env` は除外されます)
2. https://streamlit.io/cloud でリポジトリを選んで Deploy
3. 「Secrets」設定画面で `ANTHROPIC_API_KEY` を登録

これだけで、URL付きのWebアプリとして公開できます。

もう一段ステップアップしたくなったら:
- ユーザー認証を追加(例: `streamlit-authenticator` ライブラリ)
- データベースを SQLite → PostgreSQL に置き換え
- フロントを Next.js / React に作り直し、Pythonは API サーバーとして使う

そのときはまた相談してください。

## 困ったとき

| 症状 | 対処 |
| --- | --- |
| `SEED起動.bat` が一瞬で消える | コマンドプロンプトで `cd C:\自作アプリ\SEED` してから `SEED起動.bat` を実行するとエラーが見える |
| `python` コマンドが見つからない | ステップ1のPATH設定を確認 |
| `pillow` のビルドが失敗する | Python 3.14 で動いている可能性。3.12 をインストール後、`venv` フォルダを削除して再実行 |
| `ANTHROPIC_API_KEY が設定されていません` | `.env` ファイルが正しい場所にあるか、キーがセットされているか確認 |
| 環境がよく分からない状態になった | `診断ツール.bat` をダブルクリックして状態確認 |
